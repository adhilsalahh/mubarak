# Public Assets

Add the following images to this folder for the website:

- `kitchen-banner.jpg` - Main banner image of a luxury kitchen
- `modular-kitchen.jpg` - Image for modular kitchen service
- `custom-kitchen.jpg` - Image for custom kitchen service
- `luxury-kitchen.jpg` - Image for luxury kitchen type
- `smart-kitchen.jpg` - Image for smart kitchen type
- `normal-kitchen.jpg` - Image for a standard kitchen design
- `vision-2030.jpg` - Image representing Saudi Vision 2030
- `kitchen-pattern.png` - Subtle pattern overlay for banner (optional)

You can find high-quality kitchen images from:
- Unsplash.com
- Pexels.com
- Pixabay.com

Make sure images are optimized for web (under 500KB each) and have good resolution (at least 1920x1080 for banner).
